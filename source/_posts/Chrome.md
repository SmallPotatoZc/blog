# 内置控制台的使用
1. 元素选择器
```
$ // 相当于 document.querySelector 。  
$$ // 相当于 document.querySelectorAll 。  
$_ // 返回上一个表达式的值  
$0-$4 // 返回最近5个Elements面板选中的DOM元素。  
dir //  console.dir  
keys // 取对象的键名, 返回键名组成的数组  
values // 去对象的值, 返回值组成的数组
方向键盘的上下键
copy(content)  // 将content（也可以说是content）复制到剪贴板
```
